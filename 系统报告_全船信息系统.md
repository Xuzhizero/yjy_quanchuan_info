## 七、碰撞预警系统 (COLREGS)

### 7.1 预警算法 (`alarm.py` + `algo_utility.py`)

- **船舶领域模型 (Ship Domain)**: 基于船长的椭圆领域
  - 前方: 9.9 × 船长 × scale
  - 后方: 9.9 × 船长 × scale  
  - 左舷: 1.5 × 船长 × scale
  - 右舷: 1.05 × 船长 × scale

- **碰撞态势分类**: 对遇(HEADON)、交叉(CROSSING)、追越(OVERTAKING)、直航(STANDON)

- **警报等级 (Alarmstufe)**:
  - 0 = 安全
  - 1 = 注意
  - 2 = 危险
  - 3 = 紧急(系统降级为2，可触发紧急停车)

- **数据来源**: DCPA(最近会遇距离) + TCPA(最近会遇时间) + 船舶领域侵入判断

### 7.2 超时检测机制

- 每5秒检查各通道最后收数时间
- 超过10秒无数据则报告丢失：
  - 雷达丢失: `LOST:lada` → 0
  - GPS丢失: `LOST:GPS` → 0

### 12.1 系统启动顺序

```bash

# 3. 启动 HaiTub Node.js 后端 (如未自启)

# 4. 启动全信中枢 (终端2)
cd /home/ship6081/20260112全信/ && python3 ship_info_system.py

# 5. 启动 ROS2 控制节点 (终端3)
source /home/ship6081/control_ws/install/setup.bash
ros2 launch startup node_launch.py
```
